# 🛡️ Zero-Trust Shield AI - Network Intrusion Detector

AI-powered cybersecurity system that detects suspicious network behavior and predicts possible intrusions using Machine Learning.

## Features
- AI intrusion detection
- Network traffic risk analysis
- FastAPI security API
- Streamlit monitoring dashboard
- ML model training pipeline
- Simulated real-time traffic monitoring

## Installation

pip install -r requirements.txt

## Train Model

python train_model.py

## Run Dashboard

streamlit run dashboard.py

## Run API

uvicorn api:app --reload
